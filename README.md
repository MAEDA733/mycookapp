# Mycookapp



「Mycookappとは」



・飲食店のプロである料理人が考えた家庭的なレシピを一般のユーザーが閲覧できるサービスです。


　プロの味付けテクニックや簡単時短料理が投稿されることでユーザが抱える毎日の晩御飯に悩む問題等を解決します。
